== Description ==

The WP Mailinglijst plugin allows you to quickly and easily add a signup form for your Mailinglijst list. 

After installation you can enter your mailinglijst 'lijstnummer' on the setup page. The plugin supports popup, iframe and FAST versions of the sign-up form.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your 'lijstnummer' in the settings page under 'Setting/Mailinglijst'

Now you can add a [mailinglijst] shortcode inside your posts!
